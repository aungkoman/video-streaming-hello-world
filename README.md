# Video Streaming.

## setup

```
npm install
node server.js

# localhost:3000
# localhost:3000/watch
```



## Prerequest

- [ ] install ffmpeg in host computer / server from https://www.ffmpeg.org/
- [ ] verify ffmpeg installation by enter "ffmpeg" in terminal
